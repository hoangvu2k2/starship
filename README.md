## Requirements

```shell
node = ^16.13.0
yarn = 1.22.19
```

## Description
REST API to connect to the [Star Wars API](https://swapi.dev/documentation#intro)
This application was built with Nest.Js (https://nestjs.com/)

## Installation

```bash
$ yarn install
```

## Running the app

Copy env.local and remove it to .env 
Run the following:

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Test

```bash
# unit tests
$ yarn run test
```

## Try it  

Launch Swagger UI and try these APIs 

```
http://localhost:3000/api

```
